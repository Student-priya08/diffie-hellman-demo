# Diffie-Hellman Key Exchange (Python)

This project demonstrates a simple implementation of the Diffie-Hellman Key Exchange algorithm using Python.

## Description

The program simulates two parties (Alice and Bob) generating public keys and computing a shared secret key.

## How to Run
python3 Diffie_Hellman.py

## Purpose

Educational demonstration of cryptographic key exchange.

-Not intended for real cryptographic security.
